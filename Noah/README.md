# REUNION ISLAND SIMULATOR

---

## Contexte

 - Bienvenue ! Ce futur jeu est un jeu à choix qui raconte l'histoire d'une personne à La Réunion, de sa naissance jusqu'à sa mort. Chaque décision que vous prenez aura des conséquences, alors soyez intelligent ! ;D


 - Le **but** est de survivre le plus longtemps possible et, éventuellement, de réussir votre vie.

## Usage

```python project.py```

 - Ce "jeu" utilise [Tkinter](https://docs.python.org/fr/3/library/tkinter.html) afin de donner une interface digne de ce nom

- Tkinter devrait être par installé défaut, si ce n'est pas le cas, lancez la commande suivante:

```sudo apt-get install python-tk```


### Entrée

- Pour jouer, vous devez appuyez sur les boutons qui sera disponible a l'ouverture de l'interface

### Sortie

 - Une fois un choix effectué, les options disponibles se mettront à jour et vous pourrez continuer votre progression.
