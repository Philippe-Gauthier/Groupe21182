# Stranded - Une histoire dont vous etes le heros #

## But du projet ##

Faire une histoire dont vous etes le heros.

---

## Histoire generale ##

Vous etes partis en vacance avec vos amis. Soudainement, le bateau de croisiere sur lequel vous etes recoit un impact, ce qui le fends en deux... Tenterez-vous de survivre, et si oui, comment...

---

## Entrees ##

Les entrees attendues sont les choix faits par l'utilisateur par l'intermediaire d'un bouton dans une fenetre

---

## Sorties ## 

La sortie sera la prochaine page de l'histoire, ainsi que les prochaines options de choix.

---

## Structure de controle ##

La structure de controle sera un arbre, donc elle sera lineaire. Cependant, je fais un arbe 3D, qui prends en compte une dimension du temps.
