# Description
- Choice2forge est un jeu qui permet de tester votre connaissances dans le monde de technologies, plus précisament sur le côté Hardware des ordinateurs.
# Execution
- Pour lancer le jeu utilisé la fonction Run Python File en VS code ou double clicker le ficher python du Jeu.
# Entrée 
- Les entrées de la fonction vont être Question, choix1, choix2, choix3, et correct answer.

# Sortie
- Le résultat affiché va être celui donner par retourne aprés toute les test fait par le programme précis pour le choix de réponse choisi.

# Structure de contôle
-  Le jeu affiche une question avec plusieurs choix de réponse, attend la saisie de l'utilisateur, puis utilise des conditions pour vérifier la réponse et afficher le message correspondant.

# Théme (Choice2Forge)
- C'est un jeu à choix multiple concernant le côté technologique comme les ordinateurs, les téléphone, leurs composantes et même les marques qui le fabrique.
