# Entrée 
- Les entrée du fonction vont-être Question, choix1, choix2, choix3, et correct answer.

# Sortie
- Le résultat affiché va être celui donner par retourne aprés toute les test fait par le programme précis pour le choix de réponse choisi

# Structure de contôle
- La question les choix et la logic de réponse vont être tout faite dans une définition. Pour que les choix soit affiché on va utilisé Print(). Pour la question on utilise seulement une variable nommé question pour qu'on met la question qu'on veut lorsque on appelle la fonction quiz qui est la définition, Input va-être utilisé pour que l'utilisateur choisi sa reponse, une variable de valide choices va être utilisé pour que s'il contredit la réponse le programme va affiché mauvaise réponse on va utilisé elif pour ça, if va être utilisé pour que si la réponse de l'utilisateur égale la variable de correct answer elle va affiché bonne reponse, else c'est une méthode pour protéger mon programme si l'utilisateur répond avec un choix non valide, et return decision va être utilisé pour retourné la réponse valide au cas.

# Théme (Choice2Forge)
- C'est un jeu à choix multiple cocernant le côté technologique comme les ordi, les telephone, leurs composantes et même les marques qui le fabrique.
